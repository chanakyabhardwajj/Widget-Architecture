Widget Architecture Mock

After reading up a lot about modular approach, I have tried my hand at building up this small demo.
The most important lesson from this exercise has been the forced separation-of-concern in the code (all thanks to AMD/RequireJS).
This surely is a fun and scalable way to build large applications.


This a project illustrating the importance of self-contained modules for widgets.
The following 2 modules : Text & News are independent of each other and embedded into this host page.

Text Widget

News Widget


The project utilises mockjax & chanceJS to mock the api-services (this keeps the demo portable and contained).
